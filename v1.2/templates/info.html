Stockings Paper Trading is a simulated, simplified trading environment. There is no real risk involved in trading on this website.

The term "paper trading" is a reference to a time when aspiring traders would practice on paper before taking on real risk in live markets.
Some features you won't find on this website:
  • Bid/ask/spread
  • limit buy/sell
  • options contracts
  • futures contracts

Investopedia is a fantastic resource if you're interested in learning more about investing.

This website was made just for practice, and is intended to showcase compentency in the following areas:

  • Back-end web development -- Python, JS
  • Network architecture -- Flask
  • Database management -- SQL
  • Front-end web development -- HTML, CSS
  • Preprocessor scripting languages -- SASS
  • API usage -- Yahoo Finance, NASDAQ
  • XML RSS feed and JSON data parsing
